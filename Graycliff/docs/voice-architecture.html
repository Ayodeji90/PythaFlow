A turn, in one breath

why it feels human
LISTENINGVAD watches for end of speech → THINKINGSTT final → LLM streams, tools fire → SPEAKINGsentence-chunked TTS streams out ↻
BARGE-IN — the guest talks over the agent → VAD fires → gateway cancels the LLM stream and flushes the TTS queue within one audio frame (~20 ms). The agent stops talking when interrupted; that single behavior is most of what makes it feel human.

Build vs. buy

decision record
✓

Cascaded streaming pipeline, our gateway

Full control of brand voice, prompt, tools, and data; each stage swappable; ~3–6× cheaper per session than managed platforms; the same gateway later answers the phone line.

✕

Managed voice-agent platforms (Vapi, Retell, Bland)

Lock-in at the whole-agent level — the exact dependency we just architected away. Per-minute pricing erodes margin; guest audio transits a third party.

◆

Realtime speech-to-speech APIs

Kept as one more provider wrapper behind the same session interface. We can A/B it for latency; we never depend on it.
