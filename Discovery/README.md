# Discovery Phase (Phase 1)

## Objective
Identify the most valuable, AI‑solvable pain point for small‑to‑medium restaurants. Validate the problem with real owners/managers and assess willingness to pay.

## Steps
1. **Select interview targets** – 5‑7 local restaurants (different sizes, cuisines, service models).
2. **Schedule interviews** – 30‑45 min each, in‑person or video call.
3. **Conduct interview** using the questionnaire below.
4. **Synthesize findings** – list recurring pain points, estimate impact (time/money saved), and rank by AI feasibility.
5. **Select MVP problem** – the highest‑impact, easiest‑to‑prototype AI solution.

## Deliverables
- Interview summary table
- Pain‑point matrix (impact vs. feasibility)
- Chosen MVP problem statement
- Preliminary value proposition & pricing hint
